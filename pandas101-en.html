DAY 1

Introduction to Pandas & Core Data Structures

Understand what Pandas is and learn the core data structures: Series and DataFrame.

1 What is Pandas?

Pandas is a Python library for easily working with data. It lets you handle tabular (spreadsheet-like) data just like Excel.

# Install pandas pip install pandas # Import pandas import pandas as pd print(pd.__version__)

2 Series

A Series is a one-dimensional array-like data structure. It consists of an index and values.

# Create a Series from a list s = pd.Series([10, 20, 30, 40]) print(s) # Specify custom index s = pd.Series([10, 20, 30], index=['a', 'b', 'c']) print(s['b']) # 20

3 DataFrame

A DataFrame is a two-dimensional tabular data structure. It consists of rows and columns.

# Create a DataFrame from a dictionary data = { 'Name': ['Kim Cheolsu', 'Lee Younghee', 'Park Minsu'], 'Age': [25, 30, 28], 'City': ['Seoul', 'Busan', 'Daejeon'] } df = pd.DataFrame(data) print(df) # Output: # Name Age City # 0 Kim Cheolsu 25 Seoul # 1 Lee Younghee 30 Busan # 2 Park Minsu 28 Daejeon

4 Basic Exploration Functions

Essential functions to quickly understand the structure and contents of a DataFrame.

df.head() # View first 5 rows df.tail() # View last 5 rows df.shape # (number of rows, number of columns) df.info() # Data types and other info df.describe() # Basic statistical summary df.columns # List of column names df.dtypes # Data type of each column

DAY 2

Data Selection & Indexing

Learn how to freely select rows and columns, and filter data based on conditions.

1 Selecting Columns

Various ways to select the columns you want from a DataFrame.

# Select a single column (returns a Series) df['Name'] # Select multiple columns (returns a DataFrame) df[['Name', 'Age']]

2 loc & iloc Indexing

loc is label-based indexing, iloc is integer (position)-based indexing.

# loc: label-based indexing df.loc[0, 'Name'] # 'Name' column of row 0 df.loc[0:2, ['Name', 'Age']] # Rows 0-2, Name & Age columns # iloc: integer (position)-based indexing df.iloc[0, 0] # Row 0, Column 0 df.iloc[0:2, 0:2] # Rows 0-1, Columns 0-1 df.iloc[:, 1] # All rows, Column 1

3 Conditional Filtering

Extract only the data that meets your conditions. Similar to the filter feature in Excel!

# People aged 28 or older df[df['Age'] >= 28] # Combining multiple conditions (AND: &, OR: |) df[(df['Age'] >= 25) & (df['City'] == 'Seoul')] # isin(): Select if value matches any in the list df[df['City'].isin(['Seoul', 'Busan'])] # String contains df[df['Name'].str.contains('Kim')]

4 Sorting

Sort data in ascending or descending order by a specific column.

# Sort by Age in ascending order df.sort_values('Age') # Sort in descending order df.sort_values('Age', ascending=False) # Sort by multiple columns df.sort_values(['City', 'Age'], ascending=[True, False]) # Sort by index df.sort_index()

DAY 3

Data Manipulation & Preprocessing

Learn the most commonly used real-world skills: adding/deleting data, handling missing values, and type conversion.

1 Adding & Deleting Columns

How to add new columns to a DataFrame or delete existing ones.

# Add a new column df['Score'] = [85, 92, 78] # Add a new column via calculation df['Passed'] = df['Score'] >= 80 # Delete a column df = df.drop('Passed', axis=1) # Delete a row df = df.drop(0, axis=0) # Delete row at index 0

2 Handling Missing Values (NaN)

Real-world data often contains missing values (NaN). Learn how to find and handle them.

import numpy as np # Check for missing values df.isnull() # Shows True/False df.isnull().sum() # Count of missing values per column # Drop missing values df.dropna() # Drop rows with missing values df.dropna(axis=1) # Drop columns with missing values # Fill missing values df['Age'].fillna(0) # Fill with 0 df['Age'].fillna(df['Age'].mean()) # Fill with mean value

3 Data Type Conversion

How to convert column data types. Useful when numbers are stored as strings, etc.

# Check data types df.dtypes # Convert types df['Age'] = df['Age'].astype('float') df['Score'] = df['Score'].astype('int') # String -> Date conversion df['Date'] = pd.to_datetime(df['Date'])

4 Modifying Values & apply()

Change specific values or transform data by applying functions.

# Change a specific value df.loc[0, 'Name'] = 'Kim Youngsu' # Replace values with replace() df['City'].replace('Seoul', 'Sejong') # apply() - Apply a function df['Age_Group'] = df['Age'].apply( lambda x: 'Young' if x < 30 else 'Senior' ) # map() - Dictionary mapping city_map = {'Seoul': 'Capital', 'Busan': 'Southeast'} df['Region'] = df['City'].map(city_map)

DAY 4

GroupBy & Statistical Analysis

Analyze data by groups and use various statistical functions and pivot tables.

1 GroupBy

Split data into groups based on a column and perform aggregate operations on each group.

# Sample data data = { 'Department': ['Sales', 'Engineering', 'Sales', 'Engineering', 'Sales'], 'Name': ['Kim Cheolsu', 'Lee Younghee', 'Park Minsu', 'Choi Jieun', 'Han Donghoon'], 'Salary': [5000, 6000, 4500, 7000, 5500] } df = pd.DataFrame(data) # Average salary by department df.groupby('Department')['Salary'].mean() # Multiple statistics by department df.groupby('Department')['Salary'].agg(['mean', 'sum', 'count', 'max'])

2 Statistical Functions

Various functions to compute basic statistics on your data.

df['Salary'].mean() # Mean df['Salary'].median() # Median df['Salary'].std() # Standard deviation df['Salary'].min() # Minimum df['Salary'].max() # Maximum df['Salary'].sum() # Sum df['Department'].value_counts() # Count per value df['Department'].nunique() # Number of unique values

3 Pivot Tables & Crosstabs

Summarize data just like Excel's pivot table feature.

# Pivot table pd.pivot_table( df, values='Salary', index='Department', aggfunc='mean' ) # Crosstab pd.crosstab(df['Department'], df['Position'])

4 Merging Data (Merge & Concat)

How to combine multiple DataFrames into one.

# concat: Simple concatenation pd.concat([df1, df2]) # Row-wise (vertical) pd.concat([df1, df2], axis=1) # Column-wise (horizontal) # merge: Join on a common key (similar to SQL JOIN) pd.merge(df1, df2, on='Name') # Inner join pd.merge(df1, df2, on='Name', how='left') # Left join pd.merge(df1, df2, on='Name', how='outer') # Outer join

DAY 5

Visualization & Real-World Project

Visualize data with Pandas and Matplotlib, and complete a real-world data analysis project.

1 Built-in Pandas Visualization

Pandas can easily create charts using .plot().

import matplotlib.pyplot as plt # Bar chart df['Salary'].plot(kind='bar') plt.title('Salary by Employee') plt.show() # Line chart df.plot(x='Date', y='Sales', kind='line') # Histogram df['Age'].plot(kind='hist', bins=10) # Pie chart df['Department'].value_counts().plot(kind='pie', autopct='%1.1f%%') # Scatter plot df.plot(x='Age', y='Salary', kind='scatter')

2 Matplotlib Customization

How to make your charts look even better.

# Font settings (macOS) plt.rcParams['font.family'] = 'AppleGothic' # Font settings (Windows) plt.rcParams['font.family'] = 'Malgun Gothic' # Customize chart fig, ax = plt.subplots(figsize=(10, 6)) df.groupby('Department')['Salary'].mean().plot( kind='bar', color=['#6366f1', '#ec4899'], ax=ax ) ax.set_title('Average Salary by Department', fontsize=16) ax.set_xlabel('Department') ax.set_ylabel('Salary') plt.tight_layout() plt.show()

3 File Input/Output

Load and save data in various formats including CSV, Excel, and more.

# CSV files df = pd.read_csv('data.csv', encoding='utf-8') df.to_csv('output.csv', index=False) # Excel files df = pd.read_excel('data.xlsx', sheet_name='Sheet1') df.to_excel('output.xlsx', index=False) # JSON files df = pd.read_json('data.json') df.to_json('output.json')
