DAY 1

Pandas简介 & 基本数据结构

了解Pandas是什么，学习核心数据结构Series和DataFrame。

1 什么是Pandas？

Pandas是Python中用于轻松处理数据的库。它可以像Excel一样处理表格形式的数据。

# 安装pandas pip install pandas # 导入pandas import pandas as pd print(pd.__version__)

2 Series（序列）

Series是类似于一维数组的数据结构，由索引（index）和值（value）组成。

# 用列表创建Series s = pd.Series([10, 20, 30, 40]) print(s) # 指定索引 s = pd.Series([10, 20, 30], index=['a', 'b', 'c']) print(s['b']) # 20

3 DataFrame（数据框）

DataFrame是二维表格形式的数据结构，由行（row）和列（column）组成。

# 用字典创建DataFrame data = { '姓名': ['张伟', '李芳', '王强'], '年龄': [25, 30, 28], '城市': ['北京', '上海', '广州'] } df = pd.DataFrame(data) print(df) # 结果: # 姓名 年龄 城市 # 0 张伟 25 北京 # 1 李芳 30 上海 # 2 王强 28 广州

4 基本探索函数

快速了解DataFrame结构和内容的基本函数。

df.head() # 查看前5行 df.tail() # 查看后5行 df.shape # (行数, 列数) df.info() # 数据类型等信息 df.describe() # 基本统计摘要 df.columns # 列名列表 df.dtypes # 每列的数据类型

DAY 2

数据选择 & 索引

学习如何自由选择行和列，以及根据条件筛选数据。

1 列（Column）选择

从DataFrame中选择所需列的多种方法。

# 选择单列（返回Series） df['姓名'] # 选择多列（返回DataFrame） df[['姓名', '年龄']]

2 loc & iloc 索引

loc基于标签（名称）索引，iloc基于整数（位置）索引来选择数据。

# loc: 基于标签的索引 df.loc[0, '姓名'] # 第0行的'姓名'列 df.loc[0:2, ['姓名', '年龄']] # 第0~2行，姓名和年龄列 # iloc: 基于整数（位置）的索引 df.iloc[0, 0] # 第0行，第0列 df.iloc[0:2, 0:2] # 第0~1行，第0~1列 df.iloc[:, 1] # 所有行的第1列

3 条件筛选

使用条件表达式提取所需数据。类似于Excel的筛选功能！

# 年龄大于等于28的人 df[df['年龄'] >= 28] # 多条件组合（AND: &, OR: |） df[(df['年龄'] >= 25) & (df['城市'] == '北京')] # isin(): 匹配多个值中的任意一个 df[df['城市'].isin(['北京', '上海'])] # 字符串包含判断 df[df['姓名'].str.contains('张')]

4 排序（Sorting）

按特定列对数据进行升序/降序排列。

# 按年龄升序排列 df.sort_values('年龄') # 降序排列 df.sort_values('年龄', ascending=False) # 按多列排序 df.sort_values(['城市', '年龄'], ascending=[True, False]) # 按索引排序 df.sort_index()

DAY 3

数据处理 & 预处理

学习实际工作中最常用的数据添加、删除、缺失值处理和类型转换。

1 添加 & 删除列

向DataFrame添加新列或删除现有列的方法。

# 添加新列 df['分数'] = [85, 92, 78] # 通过计算添加新列 df['是否及格'] = df['分数'] >= 80 # 删除列 df = df.drop('是否及格', axis=1) # 删除行 df = df.drop(0, axis=0) # 删除索引为0的行

2 缺失值（NaN）处理

实际数据中经常存在空值（NaN）。学习如何查找和处理缺失值。

import numpy as np # 检查缺失值 df.isnull() # 显示True/False df.isnull().sum() # 每列缺失值数量 # 删除缺失值 df.dropna() # 删除含缺失值的行 df.dropna(axis=1) # 删除含缺失值的列 # 填充缺失值 df['年龄'].fillna(0) # 用0填充 df['年龄'].fillna(df['年龄'].mean()) # 用均值填充

3 数据类型转换

转换列的数据类型的方法。当数字以字符串形式存储时特别有用。

# 查看数据类型 df.dtypes # 类型转换 df['年龄'] = df['年龄'].astype('float') df['分数'] = df['分数'].astype('int') # 字符串 -> 日期转换 df['日期'] = pd.to_datetime(df['日期'])

4 值修改 & apply()

修改特定值或应用函数来转换数据。

# 修改特定值 df.loc[0, '姓名'] = '张强' # 用replace()替换值 df['城市'].replace('北京', 'Beijing') # apply() - 应用函数 df['年龄分类'] = df['年龄'].apply( lambda x: '青年' if x < 30 else '中年' ) # map() - 字典映射 city_map = {'北京': '华北', '上海': '华东'} df['区域'] = df['城市'].map(city_map)

DAY 4

GroupBy & 统计分析

按组分析数据，运用各种统计函数和数据透视表。

1 GroupBy（分组）

按特定列将数据分组，对每组执行聚合运算。

# 示例数据 data = { '部门': ['销售', '研发', '销售', '研发', '销售'], '姓名': ['张伟', '李芳', '王强', '赵敏', '刘洋'], '年薪': [150000, 200000, 130000, 250000, 160000] } df = pd.DataFrame(data) # 按部门统计平均年薪 df.groupby('部门')['年薪'].mean() # 按部门统计多项指标 df.groupby('部门')['年薪'].agg(['mean', 'sum', 'count', 'max'])

2 统计函数

计算数据基本统计量的各种函数。

df['年薪'].mean() # 平均值 df['年薪'].median() # 中位数 df['年薪'].std() # 标准差 df['年薪'].min() # 最小值 df['年薪'].max() # 最大值 df['年薪'].sum() # 总和 df['部门'].value_counts() # 各值出现次数 df['部门'].nunique() # 唯一值数量

3 数据透视表 & 交叉表

类似Excel数据透视表的数据汇总方法。

# 数据透视表 pd.pivot_table( df, values='年薪', index='部门', aggfunc='mean' ) # 交叉表 pd.crosstab(df['部门'], df['职级'])

4 数据合并（Merge & Concat）

将多个DataFrame合并为一个的方法。

# concat: 简单拼接 pd.concat([df1, df2]) # 行方向（上下拼接） pd.concat([df1, df2], axis=1) # 列方向（左右拼接） # merge: 基于公共键合并（类似SQL JOIN） pd.merge(df1, df2, on='姓名') # 内连接 pd.merge(df1, df2, on='姓名', how='left') # 左连接 pd.merge(df1, df2, on='姓名', how='outer') # 外连接

DAY 5

可视化 & 实战项目

用Pandas和Matplotlib进行数据可视化，完成实战数据分析项目。

1 Pandas内置可视化

Pandas可以用.plot()轻松绑制图表。

import matplotlib.pyplot as plt # 柱状图 df['年薪'].plot(kind='bar') plt.title('各员工年薪') plt.show() # 折线图 df.plot(x='日期', y='销售额', kind='line') # 直方图 df['年龄'].plot(kind='hist', bins=10) # 饼图 df['部门'].value_counts().plot(kind='pie', autopct='%1.1f%%') # 散点图 df.plot(x='年龄', y='年薪', kind='scatter')

2 Matplotlib自定义

让图表更美观的自定义方法。

# 中文字体设置（macOS） plt.rcParams['font.family'] = 'Arial Unicode MS' # 中文字体设置（Windows） plt.rcParams['font.family'] = 'SimHei' # 自定义图表 fig, ax = plt.subplots(figsize=(10, 6)) df.groupby('部门')['年薪'].mean().plot( kind='bar', color=['#6366f1', '#ec4899'], ax=ax ) ax.set_title('各部门平均年薪', fontsize=16) ax.set_xlabel('部门') ax.set_ylabel('年薪（元）') plt.tight_layout() plt.show()

3 文件读写

以CSV、Excel等多种格式读取和保存数据。

# CSV文件 df = pd.read_csv('data.csv', encoding='utf-8') df.to_csv('output.csv', index=False) # Excel文件 df = pd.read_excel('data.xlsx', sheet_name='Sheet1') df.to_excel('output.xlsx', index=False) # JSON文件 df = pd.read_json('data.json') df.to_json('output.json')

4 实战迷你项目

综合运用5天所学内容分析实际数据！

# 实战项目：销售数据分析 import pandas as pd import matplotlib.pyplot as plt # 1. 读取数据 df = pd.read_csv('sales.csv') # 2. 数据探索 print(df.head()) print(df.info()) print(df.describe()) # 3. 缺失值处理 df = df.dropna() # 4. 按类别分析销售额 category_sales = df.groupby('类别')['销售额'].sum() # 5. 可视化 category_sales.plot(kind='bar') plt.title('各类别总销售额') plt.show() # 6. 保存结果 category_sales.to_csv('result.csv')
