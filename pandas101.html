DAY 1

Pandas 소개 & 기본 자료구조

Pandas가 무엇인지 이해하고, 핵심 자료구조인 Series와 DataFrame을 배웁니다.

1 Pandas란?

Pandas는 파이썬에서 데이터를 쉽게 다루기 위한 라이브러리입니다. 엑셀처럼 표(테이블) 형태의 데이터를 다룰 수 있어요.

# pandas 설치 pip install pandas # pandas 불러오기 import pandas as pd print(pd.__version__)

2 Series (시리즈)

Series는 1차원 배열 같은 자료구조입니다. 인덱스(index)와 값(value)으로 이루어져 있어요.

# 리스트로 Series 만들기 s = pd.Series([10, 20, 30, 40]) print(s) # 인덱스 지정하기 s = pd.Series([10, 20, 30], index=['a', 'b', 'c']) print(s['b']) # 20

3 DataFrame (데이터프레임)

DataFrame은 2차원 표(테이블) 형태의 자료구조입니다. 행(row)과 열(column)로 이루어져 있어요.

# 딕셔너리로 DataFrame 만들기 data = { '이름': ['김철수', '이영희', '박민수'], '나이': [25, 30, 28], '도시': ['서울', '부산', '대전'] } df = pd.DataFrame(data) print(df) # 결과: # 이름 나이 도시 # 0 김철수 25 서울 # 1 이영희 30 부산 # 2 박민수 28 대전

4 기본 탐색 함수

DataFrame의 구조와 내용을 빠르게 파악하는 기본 함수들입니다.

df.head() # 처음 5행 보기 df.tail() # 마지막 5행 보기 df.shape # (행 수, 열 수) df.info() # 데이터 타입 등 정보 df.describe() # 기본 통계 요약 df.columns # 열 이름 목록 df.dtypes # 각 열의 데이터 타입

DAY 2

데이터 선택 & 인덱싱

원하는 행과 열을 자유롭게 선택하고, 조건에 맞는 데이터를 필터링하는 방법을 배웁니다.

1 열(Column) 선택

DataFrame에서 원하는 열을 선택하는 여러 가지 방법입니다.

# 단일 열 선택 (Series 반환) df['이름'] # 여러 열 선택 (DataFrame 반환) df[['이름', '나이']]

2 loc & iloc 인덱싱

loc는 라벨(이름) 기반, iloc는 정수(위치) 기반으로 데이터를 선택합니다.

# loc: 라벨 기반 인덱싱 df.loc[0, '이름'] # 0번 행의 '이름' 열 df.loc[0:2, ['이름', '나이']] # 0~2행, 이름&나이 열 # iloc: 정수(위치) 기반 인덱싱 df.iloc[0, 0] # 0번째 행, 0번째 열 df.iloc[0:2, 0:2] # 0~1행, 0~1열 df.iloc[:, 1] # 모든 행의 1번째 열

3 조건 필터링

조건식을 사용하여 원하는 데이터만 추출합니다. 엑셀의 필터 기능과 비슷해요!

# 나이가 28 이상인 사람 df[df['나이'] >= 28] # 여러 조건 결합 (AND: &, OR: |) df[(df['나이'] >= 25) & (df['도시'] == '서울')] # isin(): 여러 값 중 하나라도 포함되면 선택 df[df['도시'].isin(['서울', '부산'])] # 문자열 포함 여부 df[df['이름'].str.contains('김')]

4 정렬 (Sorting)

데이터를 특정 열 기준으로 오름차순/내림차순 정렬합니다.

# 나이 기준 오름차순 정렬 df.sort_values('나이') # 내림차순 정렬 df.sort_values('나이', ascending=False) # 여러 열 기준 정렬 df.sort_values(['도시', '나이'], ascending=[True, False]) # 인덱스 기준 정렬 df.sort_index()

DAY 3

데이터 가공 & 전처리

실무에서 가장 많이 쓰이는 데이터 추가, 삭제, 결측값 처리, 타입 변환을 배웁니다.

1 열 추가 & 삭제

DataFrame에 새로운 열을 추가하거나 기존 열을 삭제하는 방법입니다.

# 새 열 추가 df['점수'] = [85, 92, 78] # 계산을 통한 새 열 추가 df['합격여부'] = df['점수'] >= 80 # 열 삭제 df = df.drop('합격여부', axis=1) # 행 삭제 df = df.drop(0, axis=0) # 인덱스 0번 행 삭제

2 결측값(NaN) 처리

실제 데이터에는 빈 값(NaN)이 자주 있습니다. 이를 찾고 처리하는 방법을 배웁니다.

import numpy as np # 결측값 확인 df.isnull() # True/False 표시 df.isnull().sum() # 열별 결측값 개수 # 결측값 삭제 df.dropna() # 결측값 있는 행 삭제 df.dropna(axis=1) # 결측값 있는 열 삭제 # 결측값 채우기 df['나이'].fillna(0) # 0으로 채우기 df['나이'].fillna(df['나이'].mean()) # 평균값으로 채우기

3 데이터 타입 변환

열의 데이터 타입을 변환하는 방법입니다. 숫자가 문자로 저장된 경우 등에 유용합니다.

# 데이터 타입 확인 df.dtypes # 타입 변환 df['나이'] = df['나이'].astype('float') df['점수'] = df['점수'].astype('int') # 문자열 -> 날짜 변환 df['날짜'] = pd.to_datetime(df['날짜'])

4 값 변경 & apply()

특정 값을 변경하거나, 함수를 적용하여 데이터를 변환합니다.

# 특정 값 변경 df.loc[0, '이름'] = '김영수' # replace()로 값 바꾸기 df['도시'].replace('서울', 'Seoul') # apply() - 함수 적용 df['나이_카테고리'] = df['나이'].apply( lambda x: '청년' if x < 30 else '중년' ) # map() - 딕셔너리 매핑 city_map = {'서울': '수도권', '부산': '경상권'} df['권역'] = df['도시'].map(city_map)

DAY 4

GroupBy & 통계 분석

데이터를 그룹별로 분석하고, 다양한 통계 함수와 피벗 테이블을 활용합니다.

1 GroupBy (그룹화)

특정 열을 기준으로 데이터를 그룹으로 나누고, 각 그룹에 대해 집계 연산을 수행합니다.

# 예시 데이터 data = { '부서': ['영업', '개발', '영업', '개발', '영업'], '이름': ['김철수', '이영희', '박민수', '최지은', '한동훈'], '연봉': [5000, 6000, 4500, 7000, 5500] } df = pd.DataFrame(data) # 부서별 평균 연봉 df.groupby('부서')['연봉'].mean() # 부서별 여러 통계 df.groupby('부서')['연봉'].agg(['mean', 'sum', 'count', 'max'])

2 통계 함수

데이터의 기본 통계량을 구하는 다양한 함수입니다.

df['연봉'].mean() # 평균 df['연봉'].median() # 중앙값 df['연봉'].std() # 표준편차 df['연봉'].min() # 최솟값 df['연봉'].max() # 최댓값 df['연봉'].sum() # 합계 df['부서'].value_counts() # 값별 개수 df['부서'].nunique() # 고유값 개수

3 피벗 테이블 & 크로스탭

엑셀의 피벗 테이블처럼 데이터를 요약하는 방법입니다.

# 피벗 테이블 pd.pivot_table( df, values='연봉', index='부서', aggfunc='mean' ) # 크로스탭 (교차표) pd.crosstab(df['부서'], df['직급'])

4 데이터 병합 (Merge & Concat)

여러 DataFrame을 하나로 합치는 방법입니다.

# concat: 단순 이어붙이기 pd.concat([df1, df2]) # 행 방향(위아래) pd.concat([df1, df2], axis=1) # 열 방향(옆으로) # merge: 공통 키 기준 병합 (SQL JOIN과 유사) pd.merge(df1, df2, on='이름') # 내부 조인 pd.merge(df1, df2, on='이름', how='left') # 왼쪽 조인 pd.merge(df1, df2, on='이름', how='outer') # 외부 조인

DAY 5

시각화 & 실전 프로젝트

Pandas와 Matplotlib으로 데이터를 시각화하고, 실전 데이터 분석 프로젝트를 수행합니다.

1 Pandas 내장 시각화

Pandas는 .plot()으로 간단히 차트를 그릴 수 있습니다.

import matplotlib.pyplot as plt # 막대 그래프 df['연봉'].plot(kind='bar') plt.title('직원별 연봉') plt.show() # 선 그래프 df.plot(x='날짜', y='매출', kind='line') # 히스토그램 df['나이'].plot(kind='hist', bins=10) # 파이 차트 df['부서'].value_counts().plot(kind='pie', autopct='%1.1f%%') # 산점도 df.plot(x='나이', y='연봉', kind='scatter')

2 Matplotlib 커스터마이징

차트를 더 보기 좋게 꾸미는 방법입니다.

# 한글 폰트 설정 (맥OS) plt.rcParams['font.family'] = 'AppleGothic' # 한글 폰트 설정 (Windows) plt.rcParams['font.family'] = 'Malgun Gothic' # 차트 꾸미기 fig, ax = plt.subplots(figsize=(10, 6)) df.groupby('부서')['연봉'].mean().plot( kind='bar', color=['#6366f1', '#ec4899'], ax=ax ) ax.set_title('부서별 평균 연봉', fontsize=16) ax.set_xlabel('부서') ax.set_ylabel('연봉 (만원)') plt.tight_layout() plt.show()

3 파일 입출력

CSV, Excel 등 다양한 형식으로 데이터를 불러오고 저장합니다.

# CSV 파일 df = pd.read_csv('data.csv', encoding='utf-8') df.to_csv('output.csv', index=False) # Excel 파일 df = pd.read_excel('data.xlsx', sheet_name='Sheet1') df.to_excel('output.xlsx', index=False) # JSON 파일 df = pd.read_json('data.json') df.to_json('output.json')

# 실전 프로젝트: 매출 데이터 분석 import pandas as pd import matplotlib.pyplot as plt # 1. 데이터 불러오기 df = pd.read_csv('sales.csv') # 2. 데이터 탐색 print(df.head()) print(df.info()) print(df.describe()) # 3. 결측값 처리 df = df.dropna() # 4. 카테고리별 매출 분석 category_sales = df.groupby('카테고리')['매출'].sum() # 5. 시각화 category_sales.plot(kind='bar') plt.title('카테고리별 총 매출') plt.show() # 6. 결과 저장 category_sales.to_csv('result.csv')
